# To-day

To-day is a calm, local-first daily commitment app. The active implementation is
the Expo / React Native app in `todo-today-app/`.

## Docs

- `todo-today-ios-spec.md`: product and iPhone UX source of truth
- `todo-today-expo-spec.md`: Expo / React Native implementation reference
- `todo-today-code-style.md`: code style and structure conventions
- `todo-today-app/README.md`: app commands and current implementation state
- `todo-today-app/LOCAL_DEVICE_INSTALL.md`: local iPhone install paths
